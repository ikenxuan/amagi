# amagi（泥干嘛哈哈
* 热门平台视频解析 API 基于 Node.js 的实现，支持最低node版本为 v16
* 抖音、B站

## 使用
```
npm install @ikenxuan/amagi
```
## 快速上手
正在施工中。。。
## 开发构建
> Node.js版本约定：
> v18.20.4 <= Node.js  <= v22.x.x
* 克隆仓库
```
git clone git@github.com:ikenxuan/amagi.git
```
* 安装依赖
```
pnpm install
```
* 构建
```
pnpm build
```
