# 快手 API

快手平台的所有 API 方法参考。

## Fetcher 方法

### fetchVideoWork

获取单个视频作品数据。

```ts twoslash
import amagi from '@ikenxuan/amagi'
const client = amagi({ cookies: { bilibili: '', douyin: '', kuaishou: '', xiaohongshu: '' } })
// ---cut---
const result = await client.kuaishou.fetcher.fetchVideoWork({
  photoId: '3xqxxxxxx',
  typeMode: 'strict'
})
```

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `photoId` | `string` | 是 | 作品 ID |
| `typeMode` | `'strict' \| 'loose'` | 否 | 类型模式 |

### fetchUserWorkList

获取用户公开视频列表。

```ts twoslash
import amagi from '@ikenxuan/amagi'
const client = amagi({ cookies: { bilibili: '', douyin: '', kuaishou: '', xiaohongshu: '' } })
// ---cut---
const result = await client.kuaishou.fetcher.fetchUserWorkList({
  principalId: 'KPL704668133',
  count: 12,
  pcursor: '',
  typeMode: 'strict'
})
```

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `principalId` | `string` | 是 | 用户主页 `principalId` |
| `count` | `number` | 否 | 每页数量，默认 `12` |
| `pcursor` | `string` | 否 | 分页游标，首屏可传空串 |
| `typeMode` | `'strict' \| 'loose'` | 否 | 类型模式 |

## 使用示例

```ts twoslash
const signed = kuaishouUtils.sign.signLiveApiRequest(

)
```

## 注意事项

<Callout type="warning">
快手接口需要有效的 Cookie，部分接口可能需要登录状态才能获取完整数据。
</Callout>

- 快手的 `photoId` 可以从分享链接中提取
- 快手的 `principalId` 可以从用户主页或直播间 URL 末段提取
- 评论数据可能需要登录 Cookie 才能获取
- 建议设置合理的请求间隔，避免触发风控
