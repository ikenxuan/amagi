# MCP Server

> 让 AI 助手直接访问 Amagi 文档，获得更准确的开发建议

Amagi MCP Server 让 AI 助手能够直接获取 Amagi 文档内容，在 Cursor、Claude、VS Code 等 AI 开发工具中提供更精准的代码建议和问题解答。

<Callout>
MCP Server 使用 Streamable HTTP 传输协议，部署在 Vercel 上，无需本地安装任何依赖。
</Callout>

## 快速配置

### Cursor

添加到 **Cursor Settings** → **Tools** → **MCP Servers**，或编辑配置文件：

```json title=".cursor/mcp.json"
{
  "mcpServers": {
    "amagi": {
      "url": "https://amagi-docs.vercel.app/api/mcp",
      "transport": "http"
    }
  }
}
```

也可以添加到全局配置 `~/.cursor/mcp.json`。查看 [Cursor MCP 文档](https://cursor.com/docs/context/mcp) 了解更多。

### Claude Code

添加到项目的 `.mcp.json` 配置文件：

```json title=".mcp.json"
{
  "mcpServers": {
    "amagi": {
      "url": "https://amagi-docs.vercel.app/api/mcp",
      "transport": "http"
    }
  }
}
```

添加配置后，重启 Claude Code 并运行 `/mcp` 查看连接状态。如果显示 **Connected**，说明配置成功。

查看 [Claude Code MCP 文档](https://docs.claude.com/en/docs/claude-code/mcp) 了解更多。

    "amagi": {

添加配置后，重启 Windsurf 以激活 MCP Server。

查看 [Windsurf MCP 文档](https://docs.windsurf.com/windsurf/cascade/mcp) 了解更多。

### VS Code

需要安装 GitHub Copilot 扩展，添加到项目的 `.vscode/mcp.json` 配置文件：

```json title=".vscode/mcp.json"
{
  "mcpServers": {
    "amagi": {
      "url": "https://amagi-docs.vercel.app/api/mcp",
      "transport": "http"
    }
  }
}
```

添加配置后，打开 `.vscode/mcp.json` 并点击 amagi 旁边的 **Start** 按钮。

查看 [VS Code MCP 文档](https://code.visualstudio.com/docs/copilot/customization/mcp-servers) 了解更多。

### Zed

通过命令面板打开设置（`zed: open settings`）或使用快捷键 `Cmd-,` (Mac) / `Ctrl-,` (Linux)：

```json title="settings.json"
{
  "context_servers": {
    "amagi": {
      "url": "https://amagi-docs.vercel.app/api/mcp",
      "transport": "http"
    }
  }
}
```

添加配置后，重启 Zed 并打开 Agent Panel 设置视图，检查 amagi 旁边的指示点是否为绿色。

查看 [Zed MCP 文档](https://zed.dev/docs/ai/mcp) 了解更多。

## 使用方式

配置完成后，可以向 AI 助手提问：

- "列出所有 Amagi 文档"
- "获取 SDK 使用指南"
- "搜索关于事件系统的文档"
- "如何使用 HTTP Server 模式？"
- "Amagi 支持哪些平台？"

AI 助手会自动调用 MCP Server 获取最新的文档内容来回答你的问题。

## 可用工具

MCP Server 提供以下工具供 AI 助手使用：

| 工具 | 描述 |
|------|------|
| `list_documents` | 获取所有可用的 Amagi 文档列表 |
| `get_document` | 获取指定文档的完整 Markdown 内容 |
| `search_documents` | 根据关键词搜索相关文档 |

## 故障排除

**无法连接到 MCP Server**

- 检查网络连接是否正常
- 确认 AI 工具的 MCP 配置格式正确
- 重启 AI 工具

**AI 助手没有使用 MCP Server**

- 在提问时明确提到 "Amagi 文档" 或 "查看文档"
- 某些 AI 工具可能需要手动启用 MCP 功能
