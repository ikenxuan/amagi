# 安装

<Callout type="warn">
本文档适用于 v6 预览版。安装时请使用 `@alpha` 标签。
</Callout>

## 使用包管理器安装

<Tabs items={['pnpm', 'npm', 'yarn', 'bun']}>
  <Tab value="pnpm">
```bash
pnpm add @ikenxuan/amagi@alpha
```
  </Tab>
  <Tab value="npm">
```bash
npm install @ikenxuan/amagi@alpha
```
  </Tab>
  <Tab value="yarn">
```bash
yarn add @ikenxuan/amagi@alpha
```
  </Tab>
  <Tab value="bun">
```bash
bun add @ikenxuan/amagi@alpha
```
  </Tab>
</Tabs>

## 系统要求

- Node.js >= 18.0.0
- 支持 ESM 和 CJS 两种模块系统

## 模块导入

<Tabs items={['ESM (推荐)', 'CommonJS']}>
  <Tab value="ESM (推荐)">
```ts twoslash
import amagi from '@ikenxuan/amagi'
// 或使用别名
import Client from '@ikenxuan/amagi'
```
  </Tab>
  <Tab value="CommonJS">
```js
const amagi = require('@ikenxuan/amagi').default
```
  </Tab>
</Tabs>

## 子路径导出

除了主入口，amagi 还提供了一些子路径导出：

```ts twoslash
// @noErrors
// Express 相关
import express from '@ikenxuan/amagi/express'

// Axios 实例
import axios from '@ikenxuan/amagi/axios'

// Chalk 终端颜色
import chalk from '@ikenxuan/amagi/chalk'
```

<Callout type="info">
v6 版本已移除对 log4js 的依赖，改用事件驱动架构。你可以通过监听事件来实现自定义日志。
</Callout>
