# 更新日志

## v6.0.0

### 重大变更

- **日志系统重构** - 移除 `log4js` 依赖，改用事件驱动架构
- **新增 Fetcher API** - 提供英文方法名的数据获取器
- **类型模式** - 新增 `typeMode: 'strict'` 支持严格类型返回

### 新功能

- 新增 `amagiEvents` 事件系统
- 新增 `bilibiliFetcher`、`douyinFetcher`、`kuaishouFetcher`、`xiaohongshuFetcher`
- 新增 `createBoundXxxFetcher` 工厂函数
- 新增 `TypeMode` 类型模式支持

### 废弃

- `getDouyinData`、`getBilibiliData` 等中文方法名 API 已废弃
- `logger`、`initLogger` 已移除

### 迁移

请参考 [v6 迁移指南](/docs/usage/migration-v6)
